『アクションゲームアルゴリズムマニアックス』
デモプログラムについて


■概要

　本CD-ROMには書籍『アクションゲームアルゴリズムマニアックス』のデモプログラムが収録されています。書籍で紹介したさまざまなアクションを実際に動かしてお楽しみください。また、デモプログラムの全ソースコードも同梱しています。


■構成

　デモプログラムはVisual Studio 2005用(vs2005フォルダ)とVisual Studio 2003用(vs2003フォルダ)の両方を収録しています。デモプログラムをビルドする場合には、お手持ちのVisual Studioに合ったフォルダをご利用ください。デモプログラムを実行する場合には、どちらのフォルダをご利用いただいてかまいません。

　vs2005フォルダとvs2003フォルダには、以下のサブフォルダがあります。

○Actionフォルダ
　デモプログラムです。エクスプローラなどから｢release｣あるいは｢debug｣フォルダにある｢Action.exe｣を実行してください。実行にはDirectX 9.0cと、対応ビデオカードおよびドライバ、そして各種のランタイムライブラリが必要です。詳細は書籍のStage00をご覧ください。なお、動作確認はWindows Vista/XP/2000で行いました。

○LibGameフォルダ
　DirectX関連の機能をまとめたライブラリです。

○LibUtilフォルダ
　Win32 API関連の機能をまとめたライブラリです。


■デモプログラムの起動時オプション

　環境に合わせて以下のオプションを設定すると、デフォルト(800x600)よりも高い解像度でデモプログラムを実行することができます。

-w   画面の幅
-h   画面の高さ
-fw  画面の幅(フルスクリーンモード)
-fh  画面の高さ(フルスクリーンモード)

例：
Action.exe -w 1024 -h 768 -fw 1280 -fh 1024
(ウィンドウモードで1024x768、フルスクリーンモードで1280x1024に設定)
