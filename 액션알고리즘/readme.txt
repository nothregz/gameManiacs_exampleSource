『게임 매니악스 액션 게임 알고리즘』
데모 프로그램에 대해서


■ 개요
『게임 매니악스 액션 게임 알고리즘』의 데모 프로그램과 그 소스 코드를 담고 있습니다.
서적에서 소개한 다양한 액션을 실제로 움직이고 즐기세요(자세한 조작 방법은 책의 0.8절을 참고하세요).


■ 최신본 다운로드
http://www.hanbit.co.kr/exam/2206


■ 구성
데모 프로그램은 윈도우즈 7과 비주얼 스튜디오 2013(vs2013 폴더)을 이용하여 빌드하였습니다.
그리고 부득이 이전 환경에서 작업 중이신 분들을 위해 비주얼 스튜디오 2005용(vs2005 폴더)도 함께 제공하였습니다. 2005용은 윈도우즈 XP/2000/비스타에서 빌드 및 작동을 확인하였습니다.


※ vs2013 폴더에는 하나의 파일과 네 개의 서브 폴더가  있습니다.

○ ActionGame.sln 파일
비주얼 스튜디오 2013 솔루션 파일입니다. 소스 코드를 살펴보고 직접 빌드해보려면 이 파일을 열어주세요.

○ Action 폴더
소스 코드와 리소스(그림, 폰트)를 담고 있습니다.

○ LibGame 폴더
DirectX 관련 기능을 구현한 라이브러리입니다.

○ LibUtil 폴더
Win32 API 관련 기능을 구현한 라이브러리입니다.

○ Output 폴더
데모 프로그램입니다. 탐색기 등에서 “Exe\Release" 혹은 “Exe\Debug" 폴더에 있는 "Action.exe"을 실행하세요.


※ vs2005 폴더에는 다음과 같이 3개의 서브 폴더가 있습니다.

○ Action 폴더
비주얼 스튜디오 2005용 솔루션 파일, 소스 코드, 리소스, 실행 파일을 담고 있습니다.
실행 파일은 Action\release 혹은 Action\debug 안의 Action.exe입니다.

○ LibGame 폴더
DirectX 관련 기능을 구현한 라이브러리입니다.

○ LibUtil 폴더
Win32 API 관련 기능을 구현한 라이브러리입니다.


■ 데모 빌드
데모 프로그램을 빌드하는 경우에는 vs2013과 vs2005 중 환경에 맞는 폴더를 이용하십시오.
빌드를 위해서는 비주얼 스튜디오 외에 DirectX SDK가 필요합니다. 마이크로소프트 다운로드 센터에서 가장 최신 버전의 SDK를 내려받아 설치해주세요.

※ 요즘은 DirectX SDK를 Windows SDK에 통합해서 배포하고 있지만, 이 책의 데모에서 참조하는 D3DX는 독립적으로 배포되는 DirectX SDK에만 포함되어 있습니다. 데모 빌드는 독립적으로 배포된 마지막 버전인 2010년 6월 버전의 DirectX SDK로 최종 검증하였습니다.


■ 데모 프로그램 실행시 옵션

다음의 옵션을 설정하면 기본(800x600)보다 높은 해상도로 데모 프로그램을 실행할 수 있습니다.

-w 화면의 폭
-h 화면의 높이
-fw 화면의 폭(전체 화면 모드)
-fh 화면의 높이(전체 화면 모드)

예:
Action.exe -w 1024 -h 768 -fw 1280 -fh 1024
(윈도우 모드로 1024x768, 전체 화면 모드로 1280x1024로 설정)


■ 사용 조건

모든 프로그램 및 데이터의 저작권은 저자인 마츠우라 켄이치로와 츠카사 유키에게 있습니다. 무단 전재/배포/변경, 무상 혹은 상용 이용, 프로그램에 포함되는 저작권 표시 및 서적 소개 수정은 금해주세요.

저작권 표시 및 서적 소개에 손대지 않는 한, 본 소스 프로그램을 다른 프로그램 작성에 이용하는 것은 허용합니다.


■ 감사의 말

데모 프로그램에서는 libpng을 사용하고 있습니다. libpng은 Glenn Randers-Pehrson 등이 저작권을 가지고 있습니다.

데모 프로그램에서는 libjpeg을 사용하고 있습니다. libjpeg는 Thomas G. Lane 등이 저작권을 가지고 있습니다.

데모 프로그램에서는 메르센·츠이스타법을 사용하고 있습니다. 이는 마츠모토 마키, 니시무라 타쿠토 등이 개발한 난수 발생 알고리즘입니다.
