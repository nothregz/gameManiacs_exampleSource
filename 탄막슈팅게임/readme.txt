『弾幕 最強のシューティングゲームを作る！』
サンプルプログラムについて


■概要

　本CD-ROMには書籍『弾幕 最強のシューティングゲームを作る！』のサンプルプログラムが収録されています。書籍で紹介したさまざまな弾幕を実際に動かしてお楽しみください。また、サンプルの全ソースコードも同梱しています。


■構成

　サンプルプログラムはVisual C++ 2008 Express Edition用です。サンプルを実行およびビルドする場合には、書籍を参考に必要なソフトウェアをインストールしておいてください。
　フォルダの構成は以下のとおりです。

○Bulletフォルダ
　サンプルプログラムの本体です。

○LibGameフォルダ
　DirectX関連の機能をまとめたライブラリです。

○LibUtilフォルダ
　Win32 API関連の機能をまとめたライブラリです。


■サンプルプログラムの起動時オプション

　サンプルプログラムはWindows Vista/XPで動作します。サンプルの実行には、DirectX 9に対応したビデオカードとドライバ、および各種のランタイムライブラリが必要です。詳細は書籍をご覧ください。
　サンプルを起動するには、エクスプローラから｢Bullet\Release｣あるいは｢Bullet\Debug｣フォルダにある｢Bullet.exe｣を実行します。なお、起動時に以下のオプションを設定すると、デフォルト(800x600)よりも高い解像度でサンプルプログラムを実行することができます。

-w   画面の幅
-h   画面の高さ
-fw  画面の幅(フルスクリーンモード)
-fh  画面の高さ(フルスクリーンモード)

例：
Bullet.exe -w 1024 -h 768 -fw 1280 -fh 1024
(ウィンドウモードで1024x768、フルスクリーンモードで1280x1024に設定)
